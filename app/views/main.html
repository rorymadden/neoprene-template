Neoprene

This is a template app showing the use of Angular, Node.js and Neo4j. It uses the neoprene library, available on npm as neoprene. This app is a simple social network manager: it lets you add and remove users and "follows" relationships between them. (Blatently copied from Node-Neo4j - credit to Aseem Kishore)

Click on a user on the left to get started or create a new one.

Enjoy playing around.
